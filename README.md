# warframe-market-parser
В этом репозитории представлена программа, позволяющая анализировать сайт warframe.market, 
визуализировать динамику цен и выбирать необходимые предметы.

## Как запустить приложение
Для запуска Prime Parser необходимо установить приложение. После установки дважды нажмите левой кнопкой мыши 
по иконке Prime Parser. 

## Как использовать приложение
Чтобы добавить какой-либо предмет для отслеживания необходимо ввести необходимые данные в соответствующие поля таблицы 
(Предмет, Тип операции, Количество, Желаемая цена) той строки записи, где стоит кнопка "Поиск". 
Когда все данные будут заполнены, нажмите левой кнопкой мыши по кнопке "Поиск". 
Если такой заказ был найден, он добавится к Вам в таблицу. 

Кнопка "Купить" копирует текст сообщения, которое Вы можете вставить в чат игры Warframe для связи с игроком.

Предмет, который Вы хотите отслеживать по цене, важно сохранить, нажав на кнопку "Добавить запись".

Удалить отслеживание предмета можно при помощи кнопки "Удалить запись", выбрав нужную запись.

## Функционал
| Функция                       | Вывод                                                                                       |
|-------------------------------|---------------------------------------------------------------------------------------------|
| collect_data_parts(*args)     | JSON файл с наиболее выгодным заказом по данным пользователя                                |
| filter_by_price(*args, price) | Отфильтрованный JSON файл                                                                   |
| apply_settings(*args)         | JSON файл с настройками, вывод данных в поля основного окна                                 |
| save_requests(*args)          | JSON файл с заказами, которые пользователь выбрал для сохранения                            |
| apply_requests(*args)         | JSON файл с заказами, возвращает актуальные данные о сохранённых заказах, заносит в таблицу |
| delete_requests(*args)        | JSON файл с заказами, удаляет выбранный заказ из таблицы и, если сохранён, из JSON файла    |
| message_parser(*args)         | Сообщение для отправки в чат                                                                |
| handle_search(*args)          | Основные данные о заказе заносит в таблицу, создаёт ссылку, которая формирует график цен    |
