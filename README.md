# warframe-market-parser

В этом репозитории представлена программа, позволяющая анализировать сайт warframe.market, визуализировать динамику цен и выбирать необходимые предметы.



\## Функционал

|Функция|Вывод|

|-------------:|-------------|

|get\_prices(\*args)|JSON файл с предметами|

|filter\_by\_price(\*args, price)|Отфильтрованный JSON файл|

|data\_visualize(\*args)|Граф динамики цен|

|notification(\*args, price)|При нахождении предложения с желаемой ценой присылает уведомление|
